# Public Assets

## Hero Image

To add the ballroom hero image:

1. Place your ballroom image file in this directory
2. Name it: `hero-ballroom.jpg` (or update the filename in `HeroSection.jsx`)
3. Recommended image specifications:
   - Format: JPG, PNG, or WebP
   - Dimensions: 1920x1080 or similar wide format
   - File size: Optimized for web (under 500KB recommended)

The image will be used as the background for the hero section on the homepage.
